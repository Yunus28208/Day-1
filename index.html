Sejarah HTML

1. Awal Mula HTML (1991)

Tim Berners-Lee menciptakan HTML di CERN untuk mengorganisir informasi di web dan memungkinkan hyperlink antar dokumen.

2. HTML 2.0 (1995)

IETF merilis HTML 2.0 dengan elemen dasar seperti formulir, tabel, dan hyperlink.

3. HTML 3.2 (1997):

W3C meluncurkan HTML 3.2 yang menambah dukungan untuk JavaScript, CSS, serta elemen baru seperti tabel dan frame.

4. HTML 4.0 (1999)

HTML 4.0 diperkenalkan dengan pemisahan konten dari presentasi melalui CSS dan dukungan untuk multimedia serta scripting, dengan konsep "strict", "transitional", dan "frameset" DTD.

5. HTML5 (2014):

HTML5 dirilis dengan integrasi teknologi seperti video, audio, grafik 2D, serta elemen semantik ( header, footer , article , section ) dan API baru seperti Web Storage dan WebSockets.

Ciri ciri dokumen HTML

1. Struktur Berbasis Tag: Dokumen HTML dibangun dengan menggunakan tag-tag yang mendefinisikan dan mengatur struktur serta format konten di halaman web. Tag HTML biasanya terdiri dari pasangan tag pembuka dan tag penutup, seperti tagname... tagname , yang membungkus konten yang ingin ditampilkan. Struktur berbasis tag ini memungkinkan browser untuk memproses dan menampilkan halaman web dengan cara yang terorganisir dan terstruktur. Setiap tag memiliki fungsi spesifik, seperti mengatur heading, paragraf, atau daftar, yang bersama-sama menciptakan halaman web yang fungsional dan dapat dibaca.

2. Penggunaan Elemen Semantik: Dengan kemajuan HTML5, elemen semantik diperkenalkan untuk memberikan makna yang lebih dalam pada struktur dokumen. Elemen-elemen semantik seperti ( header, footer, nav, dan article ) membantu dalam mendefinisikan bagian-bagian utama dari halaman web dengan cara yang lebih jelas dan terorganisir. Elemen-elemen ini tidak hanya meningkatkan pemahaman manusia mengenai struktur dokumen, tetapi juga membantu mesin pencari dan pembaca layar dalam menavigasi dan memahami konten dengan lebih baik. Elemen semantik ini memperbaiki aksesibilitas dan SEO (Search Engine Optimization), menjadikan konten web lebih mudah diakses dan diindeks.

3. Tag Pembuka dan Penutup: Setiap elemen dalam dokumen HTML dimulai dengan tag pembuka dan diakhiri dengan tag penutup. Tag pembuka, seperti tagname , menunjukkan awal dari elemen, sementara tag penutup, seperti tagname, menandakan akhir dari elemen tersebut. Konten yang diletakkan di antara tag pembuka dan tag penutup akan diproses sesuai dengan fungsi elemen tersebut. Struktur ini memastikan bahwa konten ditampilkan dengan benar dan sesuai dengan format yang diinginkan, serta membantu dalam pemeliharaan dan pembacaan kode HTML.

4. Atribut Tag: Tag HTML dapat memiliki atribut yang menyediakan informasi tambahan tentang elemen tersebut. Atribut-atribut ini dituliskan di dalam tag pembuka dan sering kali terdiri dari pasangan nama-nilai, seperti dalam contoh a href="https://example.com">Link a. Atribut ini memungkinkan penyesuaian dan peningkatan fungsionalitas elemen, misalnya menentukan URL tujuan dari hyperlink atau mengatur sumber gambar. Dengan menggunakan atribut, pengembang web dapat menambahkan fitur tambahan dan mengontrol tampilan serta perilaku elemen HTML.

5. Dukungan untuk Teks, Gambar, dan Media: HTML menyediakan dukungan luas untuk berbagai jenis media, termasuk teks, gambar, audio, dan video. Teks dapat diformat dengan berbagai elemen seperti p untuk paragraf dan h1 hingga h6 untuk heading. Gambar dapat ditambahkan menggunakan tag img, sedangkan audio dan video dapat dimasukkan dengan tag audio dan video, masing-masing. HTML juga mendukung elemen interaktif seperti formulir menggunakan tag form, memungkinkan pengguna untuk mengirimkan data melalui halaman web. Dukungan ini memungkinkan pembuatan halaman web yang kaya dan interaktif, memenuhi berbagai kebutuhan media dan konten.

6. Kemampuan Membuat Hyperlink: Salah satu fitur utama HTML adalah kemampuannya untuk membuat hyperlink yang menghubungkan halaman web atau sumber daya lainnya di internet. Dengan menggunakan tag a, pengembang dapat membuat teks atau elemen lainnya menjadi klikabel dan mengarahkan pengguna ke URL atau lokasi yang ditentukan. Misalnya, a href="https://example.com">Link a membuat teks "Link" dapat diklik untuk mengakses halaman web di URL yang ditentukan. Kemampuan ini mendukung navigasi antar halaman web dan membuat pengalaman browsing menjadi lebih dinamis dan terhubung.

Struktur dasar HTML terdiri dari

*!DOCTYPE html: Menyatakan bahwa dokumen ini adalah HTML5. *html: Elemen root yang membungkus seluruh konten halaman. *head: Bagian untuk metadata dan pengaturan dokumen. *body: Bagian yang berisi konten yang ditampilkan di browser web.
